# Finance Analytics Dashboard – Coca-Cola (KO) Analysis
## Project Overview

This project analyzes Coca-Cola (KO) financial data obtained from Yahoo Finance to evaluate financial performance, assess risk and return, and apply forecasting techniques.
The project includes financial analysis, Facebook Prophet forecasting, Monte Carlo simulation, and an interactive Streamlit dashboard for visualization and insight generation.
